Rango de tipo de entrada

input type="range" un control para ingresar un número cuyo valor exacto no es importante (como un control deslizante).
El rango predeterminado es de 0 a 100. Sin embargo, puede establecer restricciones sobre qué números se aceptan con los atributos min, maxy :step

Dependiendo de la compatibilidad del navegador: el tipo de entrada "rango" se puede mostrar como un control deslizante.
